# Выбор портов

На нашем хостинге используется специализированная защита от DDoS-атак.
В связи с этим, для корректной работы различных плагинов и сервисов, необходимо создавать порты с соответствующим типом

## Доступные типы портов

В зависимости от ваших задач, выберите соответствующий сервис при создании порта:

| Тип порта | Протокол | Для чего используется                                           |
| :--- | :--- |:----------------------------------------------------------------|
| **Minecraft Java** | TCP | Основное подключение для Minecraft Java Edition                 |
| **Minecraft Bedrock** | UDP | Для игроков с телефонов/консолей через GeyserMC                 |
| **Simple Voice Chat** | UDP | Работа одноименного плагина на голосовой чат                    |
| **PlasmoVoice** | UDP | Работа плагина PlasmoVoice                                      |
| **HTTP** | TCP | Веб-карты (Dynmap, BlueMap), веб-сервисы (Plan) и тому подобные |
| **RCON** | TCP | Для подключения через RCON                                      |

## Как создать нужный порт

1. Перейдите в раздел **Порты** в боковом меню панели управления
2. Нажмите кнопку **Создать порт**
3. В появившемся окне выберите сервис, который вам необходим
4. После создания вы увидите выделенный IP и Порт в списке
